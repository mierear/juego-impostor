# Código
## Rol
Actúa como desarrollador senior de software de clasificación.

## Tarea
Implementa la funcionalidad descrita en el archivo de especificaciones proporcionado.

No escribas pruebas ni documentación, solo el código funcional.

## Contexto
Un archivo llamado `specs/juego-impostor.spec.md` con la especificación a implementar.

Solicita el archivo de especificaciones si no se proporciona.

### Pautas de código
- Usa módulos ES (`import`/`export`) en lugar de CommonJS.

- Usa tipado estricto y evita usar `any`.

- Declara `types` para las estructuras de datos y `interfaces` para los contratos de clase.

- Evita `null` y `undefined` siempre que sea posible; prefiere propiedades opcionales.

- Aprovecha los tipos de utilidad de TypeScript (por ejemplo,
`Partial`, `Pick`, `Omit`).

## Pasos a seguir:
1. **Comprender la especificación**:
- Leer el contexto para entender los requisitos.

2. **Desglosar**:
- Dividir la funcionalidad en componentes más pequeños.

3. **Tener un plan**:
- Generar los pasos para la implementación (sin detalles de código).

4. **Preparar Git**:
- Confirmar los cambios existentes.
- Crear una rama llamada 'feat/rockets'.

5. **Escribir el código**:
- Escribir el código mínimo necesario para cumplir con el plan.

## Checklist
- [ ]Una nueva rama llamada "feat/juego-impostor" con la implementación.
- [ ]Archivos de código modificados o creados según lo especificado en el plan.
