# 🎮 Bienvenido a Juego Impostor

Tu aplicación Ionic + Angular está lista para desarrollar.

## ✨ Información del Proyecto

- **Framework**: Ionic 8.x
- **Frontend**: Angular 18.x
- **Runtime**: Capacitor 6.x (APIs nativas)
- **Node.js**: v25.9.0
- **npm**: v11.12.1

## 🚀 Pasos Rápidos

### 1. Iniciar el Servidor de Desarrollo
```bash
npm start
# o
ionic serve
```
Abre tu navegador en: **http://localhost:8100**

### 2. Crear una Nueva Página
```bash
ionic generate page pages/mi-pagina
```

### 3. Compilar para Producción
```bash
npm run build
```

### 4. Agregar Capacitor (Opcional)
```bash
# Android
npx cap add android
npx cap open android

# iOS (requiere macOS con Xcode)
npx cap add ios
npx cap open ios
```

## 📁 Archivos Importantes

- `src/app/` - Código de la aplicación
- `src/theme/variables.scss` - Estilos globales y colores
- `capacitor.config.ts` - Configuración nativa
- `angular.json` - Configuración de Angular

## 💡 Desarrollo

La aplicación viene con 3 páginas de ejemplo (Tab1, Tab2, Tab3). 
Puedes editarlas directamente en `src/app/tab*/tab*.page.ts` y `tab*.page.html`

Los cambios se reflejan en tiempo real en el navegador.
